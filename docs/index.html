Quick Start

1. Download alphaGet the latest Command-*.zip from the latest GitHub Release.
2. Move appUnzip, move Command.app to ~/Applications, then launch it.
3. Set permissionsOpen Settings -> Set Up, grant required permissions, and confirm Accessibility is green.

Most users do not need Terminal scripts. Use Install Guide for first launch, macOS permission prompts, checksum verification, and source install.

How It Works

1. Pick a promptUse built-in Compose or create custom prompt actions.
2. Attach triggersSelected text, screenshot, popup, or voice.
3. Choose deliveryExisting chat, new chat, or background command.
4. Set destinationDefault, Chat, Cowork, or Code.
5. Keep history localClipboard, dictation, and command history stay in local files; background runs use your Claude CLI setup.

Install

For alpha builds, download the latest Command-*.zip from the latest GitHub Release. Use the matching .zip.sha256 file beside the matching zip for optional checksum verification. Binary installs do not require Terminal scripts. See the Install Guide for first launch, permissions, setup checks, and source install.

Source install:

git clone https://github.com/galbutnotgirl/command
cd command
./build-agent.sh
./install-agent.sh

Local development:

./script/build_and_run.sh --verify

Optional source fallback:

./build-helper.sh

Then open Settings -> Set Up and grant Accessibility. Add Screen Recording for screenshots and Microphone for dictation.
